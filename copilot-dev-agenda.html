GitHub Copilot Developer Training
2-Day Workshop Agenda & Outline
Day 1 Foundations → Agentic → Advanced

Module 1 Copilot Dev Foundations

~90 min

Build intentional Copilot usage: mode selection, context management, instructions, models, agents, and prompts.

  1. Foundations Framing, Safety & Workflow Map
    Copilot as an acceleration layer, not a replacement for judgment. Autonomy spectrum: Completion → Ask/Chat → Plan → Agent. Safe mode choice and review boundaries.
  2. Copilot Chat Tour
    Ask, Plan, and Agent modes. Slash commands, @workspace, @terminal, #file, #selection participants.
  3. Memory, Context & Instructions
    Context window discipline and prompt quality. Repo-scoped and file-scoped instructions. Reusable prompts.
  4. Models, Agents, Skills & Customization
    Model tradeoffs: speed vs reasoning depth vs cost. Lightweight customization: custom agents and prompts.
  5. Wrap-up & Hand-off to Lab
    Q&A and transition to hands-on exercises.

🧪 Lab Exercises

  • Lab 1 — Copilot Chat Tour: Modes, Commands & CLI
    Practice Ask / Plan / Agent modes, slash commands, participants, and the Copilot CLI.
  • Lab 2 — Memory, Context & Instructions
    Create repo instructions and file-scoped instructions; observe how output changes.
  • Lab 3 — Models, Agents & Custom Prompt
    Compare models, create a custom agent, and build a reusable prompt.
Learning Objectives
  • Choose the right chat mode for the task at hand
  • Use scoped context and instructions effectively
  • Create repo-level and file-level instructions
  • Compare model, agent, and prompt tradeoffs
  • Prepare for safe, productive lab work

Module 2 Copilot Dev Agentic

~90 min

Move from assistant-style Copilot to scoped autonomy: agents, skills, background/cloud agents, instruction layering, multi-agent patterns, and debugging habits.

  1. What Is an Agent? What Is a Skill?
    The agent loop: Plan → Act → Observe → Adjust. Skills as single actions vs agents as orchestrators.
  2. Agent vs. Skill — When to Use Each
    When to delegate to an agent vs when to use a direct tool or skill.
  3. Background Agents & Cloud Agents (Coding Agent)
    Background agents for async work. Cloud coding agent for the issue → branch → PR flow.
  4. GitHub Copilot /init & Project Bootstrapping
    Using /init as a scaffold accelerator, with review required.
  5. Instruction Layering — Org, Repo & File-Scoped
    The instruction stack: org → repo → file-scoped → user → session.
  6. Multi-Agent Patterns — Squad as a Worked Example
    The multi-agent "Squad" pattern and off-ramp safety.
  7. Rubber Duck Debugging with Copilot
    Using Copilot as a reasoning partner and cross-model review strategy.
  8. Wrap-up & Hand-off to Lab
    Q&A and transition to hands-on exercises.

🧪 Lab Exercises

  • Lab 1 — Agent vs. Skill: Explore the Difference
    Compare Agent mode to Ask mode and observe behavioral differences.
  • Lab 2 — Instruction Layering: Org, Repo & File-Scoped
    Create layered instructions at each level and compare outputs.
  • Lab 3 — Build a Custom Agent & Observe Background Execution
    Create a custom review agent and observe its background behavior.
Learning Objectives
  • Explain the difference between an agent and a skill
  • Choose between direct tools and agent orchestration
  • Describe background and cloud agent workflows
  • Explain instruction layering and its precedence
  • Identify when multi-agent setups help
  • Use Copilot for reasoning and debug review

Module 3 Copilot Dev Advanced

~90 min

Advanced integration and control: MCP, APIs/plugins, memory, debugging, agentic loops, subagents, off-ramps, and ecosystem tools.

  1. MCP (Model Context Protocol) — What, Why, How
    MCP servers and approving tool calls. Configuring and consuming MCP endpoints.
  2. Using APIs & Plugins with Copilot
    API-aware workflows and specialized plugins for extended capabilities.
  3. Memory — Current State & Evolution
    Memory lifecycle: save, verify, manage. Responsible memory practices.
  4. Debugging — Chat Debug & Agent Debug Logs
    Debugging chat and agent traces. Diagnosing context issues.
  5. Agentic Loops, Subagents & Task Review
    The Ralph Loop pattern. Subagent orchestration and off-ramp design.
  6. Resources, Tools & Ecosystem
    Ecosystem overview and closing synthesis of all modules.
  7. Wrap-up & Q&A
    Final questions and day 1 close.

🧪 Lab Exercises

  • Lab 1 — Configure & Use an MCP Server
    Set up an MCP server, approve tool calls, and integrate it into a Copilot workflow.
  • Lab 2 — Debugging: Inspect Chat & Agent Logs
    Walk through debug traces to identify context and reasoning issues.
  • Lab 3 — Observe an Agentic Loop & Off-Ramp
    Trigger an agentic loop, observe its iterations, and identify off-ramp decision points.
Learning Objectives
  • Configure and use MCP servers
  • Integrate APIs and plugins into Copilot workflows
  • Manage memory responsibly
  • Debug chat and agent behavior
  • Recognize loops, subagents, and off-ramps in agentic workflows
Day 2 Hack — Business Problem Solving

Module 4 Copilot Dev Hack

~60 min

Hands-on hackathon: pick a real business problem, build a Copilot-powered agent or workflow, then demo and iterate.

  1. Business Problem Framing
    Choose a scenario and define what success looks like.
  2. Phase 1 — Planning
    Set up the project, write custom instructions, and outline the agent design.
  3. Phase 2 — Building
    Create the agent, connect tools/context, and iterate on behavior.
  4. Phase 3 — Demo & Feedback
    Present your solution, gather peer feedback, and identify improvements.
  5. Success Criteria & Next Steps
    Walk through the success checklist. Resources for continued learning.

🎯 Hack Format

Participants bring their own business problem and build a Copilot-powered solution using the skills learned in Day 1.

Learning Objectives
  • Pick a real business problem and scope it
  • Define clear success criteria
  • Build a Copilot-based solution end-to-end
  • Test and demo the solution to peers
  • Share feedback and plan next iterations
