# Presenter Notes — Module 2: Intermediate (Agentic) — Workshop Guide

## Discussion Prompts by Section

### 1. Intermediate Operating Model and Context Hierarchy (10 min)

- Which team rules should be codified in instructions this week?
- What should never be persisted to memory in your environment?
- Where do teams confuse convenience with governance?

### 2. Instructions, Memory, and Prompt Contract Design (25 min)

- Which contract element is most frequently omitted by your team?
- What instruction should be global versus folder-scoped?
- How do you measure whether prompt quality is improving?

### 3. Agent, Skill, and Loop Mechanics (25 min)

- Which workflows in your team are over-delegated today?
- What minimum evidence should an agent produce before acceptance?
- Where should verification always remain human-led?

### 4. Tool Strategy, Background/Cloud Agents, and Scaling Controls (20 min)

- Which task classes should always remain local?
- What controls must exist before cloud delegation is allowed?
- Where can `/init` reduce setup churn most for your team?

### 5. Wrap-up and Advanced Handoff (10 min)

- Which intermediate control should become a team default immediately?
- What readiness gap remains before scaling to advanced orchestration?
